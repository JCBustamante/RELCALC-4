A mapping \(F \colon {\mathbb R}^n \to {\mathbb R}^m\) is called linear if
\begin{equation*} F(\vec{x}+\vec{y}) = F(\vec{x})+F(\vec{y})\text{,} \end{equation*}
for any vectors \(\vec{x}\) and \(\vec{y}\text{,}\) and also
\begin{equation*} F(\alpha \vec{x}) = \alpha F(\vec{x})\text{,} \end{equation*}
for any scalar \(\alpha\text{.}\) The \(F\) we defined above that doubles the size of all vectors is linear. Let us check:
\begin{equation*} F(\vec{x}+\vec{y}) = 2(\vec{x}+\vec{y}) = 2\vec{x}+2\vec{y} = F(\vec{x})+F(\vec{y})\text{,} \end{equation*}
and also
\begin{equation*} F(\alpha \vec{x}) = 2 \alpha \vec{x} = \alpha 2 \vec{x} = \alpha F(\vec{x})\text{.} \end{equation*}
