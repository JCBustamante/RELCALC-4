Example 3.2.10.

Trouvons la solution générale de \(y'' + k^2 y = 0\text{,}\) pour une constante \(k > 0\text{.}\)
L'équation caractéristique est \(r^2 + k^2 = 0\text{.}\) Donc, les racines sont \(r = \pm ik\text{,}\) et, par le théorème, on a la solution générale suivante:
\begin{equation*} y = C_1 \cos (kx) + C_2 \sin (kx)\text{.} \end{equation*}
