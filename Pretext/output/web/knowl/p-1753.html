If rows of a matrix correspond to equations, it might be good to find out how many equations do we really need to find the same set of solutions. Similarly, if we find a number of solutions to a linear equation \(A \vec{x} = \vec{0}\text{,}\) we may ask if we found enough so that all other solutions can be formed out of the given set. The concept we want is that of linear independence. The same concept is useful for differential equations, for example in Chapter 3.
