Theorem 5.1.8. Alternative de FredholmNommée d'après le mathématicien suédois Erik Ivar Fredholm 3  (1866-1927)..

Exactement l'une des affirmations suivantes est vraie. Soit
\begin{equation} x'' + \lambda x = 0, x(a) = 0, x(b) = 0\tag{5.1.4} \end{equation}
a une solution non nulle, soit
\begin{equation} x'' + \lambda x = f(t), x(a) = 0, x(b) = 0\tag{5.1.5} \end{equation}
a une solution unique pour toutes les fonctions \(f\) continues sur \([a,b]\text{.}\)
