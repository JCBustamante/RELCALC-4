Our angles are always in radians. We are computing the cosine of the angle, which is really the best we can do. Given two vectors at an angle \(\theta\text{,}\) we can give the angle as \(-\theta\text{,}\) \(2\pi-\theta\text{,}\) etc., seeÂ 6.5.1. Fortunately, \(\cos \theta = \cos (-\theta) = \cos(2\pi - \theta)\text{.}\) If we solve for \(\theta\) using the inverse cosine \(\cos^{-1}\text{,}\) we can just decree that \(0 \leq \theta \leq \pi\text{.}\)
