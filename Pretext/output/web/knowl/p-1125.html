Le théorème est également vrai pour les autres types de conditions au bord que nous avons considérées. Le théorème veut dire que si \(\lambda\) n'est pas une valeur propre, l'équation non homogène (5.1.5) a une solution unique pour tous les côtés droits. Autrement dit, si \(\lambda\) est une valeur propre, alors (5.1.5) n'a pas besoin d'avoir une solution pour tous les \(f\text{,}\) et, en plus, même s'il y a une solution, la solution n'est pas unique.
