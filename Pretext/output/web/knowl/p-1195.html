On a jusqu'ici évité la question de la convergence. Par exemple, si \(f(t)\) est une vague carrée, l'égalité
\begin{equation*} f(t) = \frac{\pi}{2} + \sum_{k=1}^\infty \frac{2}{2k-1} \sin \bigl( (2k-1)\, t \bigr) \end{equation*}
tient seulement pour les valeurs de \(t\) où \(f(t)\) est continue. On n'obtient pas l'égalité pour \(t=-\pi,0,\pi\) et pour toutes les autres discontinuités de \(f(t)\text{.}\) Ce n'est pas difficile de voir que lorsque \(t\) est un entier muliple de \(\pi\) (ce qui inclut la discontinuité), alors
\begin{equation*} \frac{\pi}{2} + \sum_{k=1}^\infty \frac{2}{2k-1} \sin \bigl( (2k-1)\, t \bigr) = \frac{\pi}{2}\text{.} \end{equation*}
