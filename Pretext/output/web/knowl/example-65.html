Example 5.4.10.

De même, on traite les conditions de Neumann. On prend le problème de la valeur limite pour \(0 \lt t \lt 1\)
\begin{equation*} x''(t) + 2 x(t) = f(t)\text{,} \end{equation*}
où \(f(t) = t\) sur \(0 \lt t \lt 1\text{,}\) mais maintenant satisfaisant aux conditions aux limites de Neumann \(x'(0) = 0\text{,}\) \(x'(1)=0\text{.}\) On écrit \(f(t)\) comme une série cosinus :
\begin{equation*} f(t) = \frac{c_0}{2} + \sum_{n=1}^\infty c_n \cos (n \pi t)\text{,} \end{equation*}
où
\begin{equation*} c_0 = 2 \int_0^1 t \,dt = 1 \end{equation*}
et
\begin{equation*} c_n = 2 \int_0^1 t \cos (n \pi t) \,dt = \frac{2\bigl({(-1)}^n-1\bigr)}{\pi^2 n^2} = \begin{cases} \frac{-4}{\pi^2 n^2} \amp \text{ si } n \text{ est impair } , \\ 0 \amp \text{ si } n \text{ est pair } . \end{cases} \end{equation*}
On écrit \(x(t)\) comme une série cosinus :
\begin{equation*} x(t) = \frac{a_0}{2} + \sum_{n=1}^\infty a_n \cos (n \pi t)\text{.} \end{equation*}
On insère et l'on obtient
\begin{equation*} \begin{split} x''(t) + 2 x(t) \amp = \sum_{n=1}^\infty \Bigl[ - a_n n^2 \pi^2 \cos (n \pi t) \Bigr] + a_0 + 2 \sum_{n=1}^\infty \Bigl[ a_n \cos (n \pi t) \Bigr] \\ \amp = a_0 + \sum_{n=1}^\infty a_n (2 - n^2 \pi^2 ) \cos (n \pi t) \\ \amp = f(t) = \frac{1}{2} + \sum_{\substack{n=1\\n~\text{ impair } } }^\infty \frac{-4}{\pi^2 n^2} \cos (n \pi t) . \end{split} \end{equation*}
Donc, \(a_0 = \frac{1}{2}\text{,}\) \(a_n = 0\) pour \(n\) pair (\(n \geq 2\)) et, pour \(n\) impair, on a
\begin{equation*} a_n (2 - n^2 \pi^2) = \frac{-4}{\pi^2 n^2}\text{,} \end{equation*}
ou
\begin{equation*} a_n = \frac{-4}{n^2 \pi^2 (2 - n^2 \pi^2)}\text{.} \end{equation*}
La série de Fourier pour la solution \(x(t)\) est
\begin{equation*} x(t) = \frac{1}{4} + \sum_{\substack{n=1\\n~\text{ impair } } }^\infty \frac{-4}{n^2 \pi^2 (2 - n^2 \pi^2)} \cos (n \pi t)\text{.} \end{equation*}
