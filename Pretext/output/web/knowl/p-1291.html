On étend \(F_{\text{ impaire } }(t)\) et \(F_{\text{ paire } }(t)\) à une \(2L\)-périodique. Alors, \(F_{\text{ impaire } }(t)\) est appelée l' extension périodique impaire de \(f(t)\text{,}\) et \(F_{\text{ paire } }(t)\) est appelée l'extension périodique paire de \(f(t)\text{.}\) Pour l'extension impaire, on suppose généralement que \(f(0) = f(L) = 0\text{.}\)
