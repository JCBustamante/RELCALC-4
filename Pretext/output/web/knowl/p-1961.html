We now define the determinant recursively
\begin{equation*} \det (A) \overset{\text{ def } }{=} \sum_{j=1}^n {(-1)}^{1+j} a_{1j} \det (A_{1j})\text{,} \end{equation*}
or in other words
\begin{equation*} \det (A) = a_{11} \det (A_{11}) - a_{12} \det (A_{12}) + a_{13} \det (A_{13}) - \cdots \begin{cases} + a_{1n} \det (A_{1n}) \amp \text{ if } n \text{ is odd, } \\ - a_{1n} \det (A_{1n}) \amp \text{ if } n \text{ even. } \end{cases} \end{equation*}
