On peut analyser le comportement à long terme des solutions sans devoir les déterminer exactement. Le champ de directions de \(x' = 0.1 x(5-x)\) nous montre que:
\begin{equation*} \lim_{t\to \infty} x(t) = \begin{cases} 5 \amp \text{ si } \; x(0) > 0 , \\ 0 \amp \text{ si } \; x(0) = 0 , \\ \text{ n'existe pas ou } {-\infty} \amp \text{ si } \; x(0) \lt 0 . \end{cases} \end{equation*}
