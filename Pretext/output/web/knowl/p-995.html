Ce système d'équations linéaires est simple à résoudre. On écrit la matrice augmentée
\begin{equation*} \left[ \begin{array}{ccc|c} -1 \amp 1 \amp 1 \amp 0 \\ 1 \amp -1 \amp 0 \amp 0 \\ 0 \amp 0 \amp -1 \amp 0 \end{array} \right] \end{equation*}
et l'on effectue des opérations sur les lignes (exercice : lesquelles?) jusqu'à obtenir
\begin{equation*} \left[ \begin{array}{ccc|c} 1 \amp -1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0 \end{array} \right]\text{.} \end{equation*}
