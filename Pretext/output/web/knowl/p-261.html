Notons d'abord que \(y=0\) est une solution (singulière). Supposons donc pour la suite que \(y \not= 0\text{.}\) On sépare les variables et l'on intègre:
\begin{align*} \frac{-3}{y^2} y' \amp = x \displaybreak[0]\\ \frac{3}{y} \amp = \frac{x^2}{2} + C \displaybreak[0]\\ y \amp = \frac{3}{\nicefrac{x^2}{2} + C} = \frac{6}{x^2 + 2C}\text{.} \end{align*}
