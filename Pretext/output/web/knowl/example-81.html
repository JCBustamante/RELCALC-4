Example 6.5.5.

The vectors \((1,1)\) and \((1,-1)\) form an orthogonal basis of \({\mathbb{R}}^2\text{.}\) Suppose we wish to represent \((3,4)\) in terms of this basis, that is, we wish to find \(a_1\) and \(a_2\) such that
\begin{equation*} (3,4) = a_1 (1,1) + a_2 (1,-1)\text{.} \end{equation*}
We compute:
\begin{equation*} a_1 = \frac{\langle (3,4), (1,1) \rangle}{ \langle (1,1), (1,1) \rangle } = \frac{7}{2}, \qquad a_2 = \frac{\langle (3,4), (1,-1) \rangle}{ \langle (1,-1), (1,-1) \rangle } = \frac{-1}{2}\text{.} \end{equation*}
So
\begin{equation*} (3,4) = \frac{7}{2} (1,1) + \frac{-1}{2} (1,-1)\text{.} \end{equation*}
