Encore une fois, \(A\) ne peut pas être nul si \(\lambda\) est une valeur propre, et \(\sin ( \sqrt{\lambda}\, \pi)\) est identiquement nul lorsque \(\sqrt{\lambda} = k\text{,}\) où \(k\) est un nombre entier positif. Donc, les valeurs propres positives sont encore \(k^2\) pour tous les entiers \(k \geq 1\text{,}\) et les fonctions propres correspondantes peuvent être prises comme \(x=\cos (k t)\text{.}\)
