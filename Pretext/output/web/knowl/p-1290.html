On prend une fonction \(f(t)\) définie sur \([0,L]\text{.}\) Sur \((-L,L]\text{,}\) la fonction est définie.
\begin{align*} F_{\text{ impaire } }(t) \amp \overset{\text{ d\'ef } }{=} \begin{cases} f(t) \amp \text{ si } \; 0 \leq t \leq L ,\\ -f(-t) \amp \text{ si } \; {-L} \lt t \lt 0 , \end{cases}\\ F_{\text{ paire } }(t) \amp \overset{\text{ d\'ef } }{=} \begin{cases} f(t) \amp \text{ si } \; 0 \leq t \leq L ,\\ f(-t) \amp \text{ si } \; {-L} \lt t \lt 0 . \end{cases} \end{align*}
